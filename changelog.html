1.6

  • New Themes! The theme manager has been opened up publicly and has a brand new theme to skin the forums with! This will default to on but can be turned off by going into the config.
    • Furthermore, more themes will be added later that do not require an update to get. Simply refresh the themes menu to get the latest and greatest!
  • New Moderator tools! Added some new moderator tools to help moderate the forums in regards to the spam. If you are a moderator, message StrangeOne101 to gain access to this.
  • Light cleanups. Removed some images for the old themes as they are no longer used.
  • Theme Database change. Switched from using pastebin to fetch themes to strangeone101.github.io (my pastebin got full lol)
